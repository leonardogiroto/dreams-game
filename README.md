# InteractiveGame

An interface and tool to allow playing with friends in a remote manner one of my favorite board games.

## Environment

React 16.13.1   
Typescript 3.7.5  
Material UI 4.11.0   
Firebase 7.14.2  

## Installation

```
npm install
```

## Running the app

```
npm run start
```

## Building the app

```
npm run build
```
